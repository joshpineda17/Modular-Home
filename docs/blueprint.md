# **App Name**: EcoHabitat Roadmap

## Core Features:

- Informational Landing Page: Presents the vision of the future of sustainable housing
- Roadmap Visualization: Interactive display of the 3-horizon roadmap (bamboo, modular homes, biofabrication).
- Material Comparison Tool: Compares the carbon footprint and structural properties of different building materials; this tool will guide the LLM whether to use certain materials
- HAUS Policy Information: Details on the HAUS policy incentives for sustainable building practices.
- Modular Home Configurator (Simple): A basic tool to visualize modular home design possibilities.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to represent nature and sustainability.
- Background color: Light beige (#F5F5DC) for a natural and calming feel.
- Accent color: Warm brown (#A0522D) for highlighting key information.
- Headline font: 'Alegreya', a serif with an elegant, intellectual, contemporary feel.
- Body font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality.
- Use icons related to nature, architecture, and technology to represent different aspects of the ecosystem.
- One-page 'long scroll' design to linearly guide the user through the roadmap.
- Subtle transitions and animations when scrolling between sections to maintain user engagement.
